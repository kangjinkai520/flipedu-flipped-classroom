# flip-classroom-uniapp

HBuilderX 可直接打开的 uni-app 前端骨架。

## 已搭好的页面

- 登录
- 首页
- 管理员用户管理
- 教师课程、课程成员、教学资料、课程通知、课堂签到、随堂考试
- 学生课程、教学资料、课程通知、课堂签到、随堂考试

## 默认后端地址

见 `common/config.js`：

- `http://127.0.0.1:18081`

如果手机真机调试，请改成你电脑的局域网 IP。

## 建议打开方式

1. 用 HBuilderX 打开本目录
2. 先启动后端 `flip-classroom-backend`
3. 运行到 H5 或微信小程序

## 演示账号

- 管理员：`admin01`
- 教师：`teacher01`
- 学生：`student01`

密码使用你当前数据库中的演示密码。
